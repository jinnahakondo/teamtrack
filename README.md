# TeamTrack - Smart Project & Task Collaboration System

A full-stack team collaboration and task management system using Next.js, MongoDB, and NextAuth. It supports user signup/login, role-based access control, projects and tasks, team member assignment, comments and notifications, dashboard analytics, and activity logs.

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Set up environment variables
cp .env.example .env.local
# Edit .env.local with your MongoDB URI and NextAuth secret

# 3. Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
